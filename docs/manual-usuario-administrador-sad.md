## 1. Introducción
Este manual describe el uso del Sistema Integrado de Archivos Digitales (SAD) para la gestión documental institucional: digitalización (OCR), versionamiento, flujos de firma digital y auditoría. Está dirigido a usuarios finales y administradores.

## 2. Roles y permisos
- Administrador: gestiona usuarios, roles y parámetros del sistema; accede a todos los módulos.
- Editor/Digitalizador: registra y mantiene documentos, ejecuta OCR, inicia flujos de firma.
- Lector: consulta y descarga documentos y reportes autorizados.

## 3. Requisitos de acceso
- Navegador actualizado (Chrome/Edge/Firefox).  
- Credenciales institucionales asignadas.  
- Conexión a la red institucional o VPN, según política.

## 5. Módulos principales
### 5.1 Gestión de documentos
- Crear documento: “Nuevo documento” → completar metadatos → adjuntar archivo(s).
- Digitalización (OCR): adjuntar imagen/PDF escaneado → ejecutar OCR → validar y guardar texto extraído.
- Versionamiento: “Nueva versión” para reemplazos o actualizaciones; conserve historial.
- Búsqueda y filtros: por metadatos, fechas, estado de firma y responsable.

### 5.2 Flujos de firma digital (Firma Perú)
- Iniciar flujo: seleccionar documento → “Enviar a firma” → definir secuencia de firmantes → confirmar.
- Seguimiento: revisar estado (pendiente, en proceso, firmado, rechazado); recibir notificaciones.
- Reversión: ante rechazo, aplicar correcciones y reintentar flujo.

### 5.3 Auditoría y reportes
- Bitácora: consulta de eventos (creación, edición, firma, descargas).
- Reportes: exporte listados por periodo, usuario o estado.

### 5.4 Administración (solo Administrador)
- Usuarios: crear/editar/bloquear usuarios, restablecer contraseñas.
- Roles: asignar y revisar permisos; auditoría de accesos.
- Parámetros: política de contraseñas, tiempos de sesión, configuración de notificaciones.

## 6. Procedimientos
### 6.1 Registrar y digitalizar un documento
1. Nuevo documento → metadatos (título, tipo, unidad, etiquetas).  
2. Adjuntar archivo escaneado.  
3. Ejecutar OCR → revisar/editar texto → guardar.  
4. Asignar responsables y estado.

### 6.2 Generar una nueva versión
1. Abrir documento → “Nueva versión”.  
2. Adjuntar archivo actualizado → describir cambios → guardar.  
3. Verificar historial de versiones.

### 6.3 Iniciar un flujo de firma
1. Abrir documento → “Enviar a firma”.  
2. Seleccionar firmantes y orden → enviar.  
3. Monitorear estado hasta “Firmado”.

### 6.4 Consultar auditoría
1. Módulo Auditoría → filtros (usuario, rango de fechas, acción).  
2. Exportar resultados si es necesario.

### 6.5 Administrar usuarios y roles (Administrador)
1. Administración → Usuarios → “Nuevo”.  
2. Completar datos y asignar rol.  
3. Revisar accesos trimestralmente.

## 7. Buenas prácticas y seguridad
- Mantenga credenciales bajo custodia; cambie contraseñas periódicamente.  
- Revise y actualice roles ante cambios de personal.  
- No comparta documentos sensibles fuera de los canales institucionales.  
- Verifique el estado de firma antes de difundir documentos.
